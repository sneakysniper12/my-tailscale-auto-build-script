# my-tailscale-auto-build-script
the script i built and use to have a simple one line install script for linux


THE ONE LINE INSTALL COMMAND:

curl -fsSL https://raw.githubusercontent.com/sneakysniper12/my-tailscale-auto-build-script/main/install-tailscale.sh \
| AUTH_KEY="your auth key" API_KEY="your API key" sudo bash
